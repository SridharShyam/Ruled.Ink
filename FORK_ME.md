# How to Make Ruled.Ink Yours

## 1. Fork the repo
```bash
git clone https://github.com/YOUR_USERNAME/Ruled.Ink.git
cd Ruled.Ink
```

## 2. Install dependencies
```bash
npm install
```

## 3. Personalize
Open `src/data/user-config.js` and edit:
- Your name
- Your wake time, college/work hours, commute duration
- Your skills (name, current topic, starting progress %)
- Your default tasks
- Your projects
- Your content posting targets

## 4. Run locally
```bash
npm run dev
```

## 5. Deploy to Vercel (free)
```bash
npm install -g vercel
vercel
```
Or click the **Deploy** button in the `README.md`.

## 6. Optional — update the favicon
Replace `public/favicon.svg` with your own icon.

That's it. The rest of the app works out of the box.
