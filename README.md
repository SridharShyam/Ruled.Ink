# Ruled.Ink

> Your personal productivity system. Built like a notebook. Works like a system.



## What is Ruled.Ink?
Ruled.Ink is an open-source personal OS for students and builders.
It combines a daily schedule tracker, skill progress system, content 
pipeline, focus timer, project tracker, and activity log — all in one 
calm, notebook-inspired interface.

Built because productivity apps are either too complex (Notion) 
or too simple (a to-do list). This is the middle ground.

## Features
- **Live Daily Timeline**: With "Now Playing" block detection.
- **Skill Tracker**: Monitor progress across 6 domains with visual progress bars.
- **Content Pipeline**: Kanban board for YouTube, LinkedIn, and Instagram.
- **Focus Mode**: Integrated Pomodoro timer with daily session tracking.
- **Project Board**: Manage missions with GitHub and live deployment links.
- **Content Calendar**: Track posting frequency against custom targets.
- **Commute Mode**: Mobile-optimized, one-thumb layout for on-the-go focus.
- **Weekly Planner**: Drag-and-drop planning with Google Calendar (.ics) export.
- **Activity Log**: Persistent historical log of all your system interactions.
- **Warm Paper Theme**: A custom cream/sage aesthetic designed for focus.
- **100% Local**: No backend, no accounts, and no tracking. All data lives in your browser.

## Tech Stack
React · Vite · Tailwind CSS · localStorage · Deployed on Vercel

## Fork & Personalize in 5 Steps
1. **Fork** this repo to your account.
2. **Clone** it locally to your machine.
3. Open `src/data/user-config.js`.
4. **Edit** your name, schedule, skills, and default tasks.
5. Run `npm install` then `npm run dev` to see your personalized OS.

See `FORK_ME.md` for the full guide including Vercel deployment.
